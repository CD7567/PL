Квантовый осциллятор
E_n = h[omega](n + 1/2) -- дискретная энергия осциллятора, n in N_0
h -- редуцированная постоянная Планка

Самая низкая энергия соответствует основному колебательному состоянию (энергии эквидистантны)

Сколько молекул в каком состоянии? -- Распределение Гиббса (нормальное по энергии)

w_i sim exp(-E_i/kT) -- формула распределения Гиббса
LRArrow
w_i = (1/z)(exp(-E_i/kT)), где sigma w_i = 1
RArrow z = sigma exp(-E_i/kT) -- статистическая сумма

Для осциллятора
z = exp(-h[omega]/2kT) + exp(-3h[omega]/2kT) + ... + exp(-(1+2n)h[omega]/2kT) + ... = exp(-h[omega]/2kT) (1 + exp(-h[omega]/kT) + ...) = <сумма геометрической прогрессии> =
  = exp(-h[omega]/2kT) 1/(1 - exp(-h[omega]/kT))

Энергию берем с точностью до константы, физический смысл сохраняется.
Самая низкая энергия осциллятора -- нулевая энергия колебаний. (Следует из уравнения Шредингера)

Средняя энергия осциллятора?
В сущности, энергия -- случацная величина. При этом w -- вероятность, тогда считаем среднюю энергию, как матожидание
E_mid = sigma E_i w_i = <другим способом> = - partial(ln z)/partial([beta]), [beta] = 1/kT
RArrow
h[omega]/(exp(h[omega]/kT) - 1)

Теплоемкость на квантовый осциллятор?
c_v = dE_mid/dT -- просто дифференцируем предыдущую формулу

Th [О равнораспределении энергии по степеням свободы]
В общем случае N атомов в молекуле. -- 3N степеней
-- Поступательных 3
-- Вращательных в нелинейной? 3 || В линейной? 2, т.к вырождается
-- Колебательных? В нелинейной 3N - 6 || 3n - 5

Сколько на поступательную?
kT/2

Сколько на вращательную?
kT/2

Сколько на колебательную?
kT (т.к два вида энергии -- кинетическая и потенциальная)


При больших температурах
E_mid approx h[omega]/((h[omega])/kT) = kT -- То есть при маленьких температурах колебательные степени заморожены

Характеристическая колебательная температура
[theta] def= h[omega]/k => T >> [theta] => kT
                           T << [theta] => По формуле E_mid

Для некоторых молекул

[theta][кислород] = 2200К

При этом при таких температурах молекулы могут диссоциировать из-за превышения энергии связи
